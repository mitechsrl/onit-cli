# Onit-cli
Onit development CLI

Questo pacchetto fornisce il comando **onit**, utility per la gestione di progetti NodeJS basati su onit.

## Prerequisiti
E' richiesto l'uso di Node 12 o superiore

## Installazione
```shell
npm install -g @mitech/onit-cli
```
